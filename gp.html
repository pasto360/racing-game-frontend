🏁 Prima configurazione

I dati vengono salvati online su JSONBin.io (gratuito, nessun database). Bastano 2 minuti per configurarlo — poi tu e il tuo amico vedrete sempre tutto aggiornato da qualsiasi dispositivo.

Passo 1 — Crea account gratuito
Vai su jsonbin.io, registrati e vai su Account → API Keys. Copia la tua Master Key.
Passo 2 — Incolla la API Key
La trovi in jsonbin.io → Account → API Keys → Master Key
Passo 3 — BIN ID (solo se hai già dati)
Prima volta? Lascia vuoto — viene creato in automatico.
Hai già un BIN? Incolla l'ID per ricaricare i tuoi dati.
Passo 4 — Nomi e campionato
💡 Da condividere con il tuo amico: dopo la configurazione, vai su ⚙ Impostazioni e copia il BIN ID. Il tuo amico dovrà inserire la stessa API Key e lo stesso BIN ID al primo accesso.
